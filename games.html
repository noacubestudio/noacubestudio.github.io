I made some puzzle games, generally inspired by Sokoban. You can play them on itch.io.
The games currently published feature a single level each and require rule discovery. They were made using PuzzleScript and run in the browser. I would recommend the latest one, >.

I spent a while working on an unreleased game that is more complex and might be developed in some other engine or language. For now, I recommend the game Can of Wormholes (Steam), made by munted finger. It is a puzzle game that released in the meantime with a similar concept to the one I was working on and is just brilliant!
